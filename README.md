## Setup
+ To run this project, using PostgreSQL and npm, install the seeded database and server:
  ```
  $ cd server
  $ npm run quick_install
  ```
+ Then using your browser open the landing page:
  ```
  $ cd ../client
  $ open index.html
  ```
